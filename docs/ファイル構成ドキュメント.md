## プロジェクト概要

相続手続きを一元管理するWebプラットフォーム。案件管理・タスク管理・スケジュール・請求入金・ドキュメント管理・レポートの6つの業務機能を提供し、Claude APIによるAI書類生成にも対応。

---

## 技術スタック

| 技術 | 用途 |
|------|------|
| Next.js 16 (App Router) | フロントエンド + サーバーサイド |
| React 19 | UIライブラリ |
| TypeScript | 型安全性 |
| Tailwind CSS 4 | スタイリング |
| Supabase | PostgreSQL DB + 認証 + Storage |
| Anthropic Claude API | AI書類生成 |
| Render | ホスティング・デプロイ |

### API Route

| ファイル | 内容 |
|----------|------|
| `src/app/api/ai-document/route.ts` | **AI書類生成API**（POST `/api/ai-document`）。リクエストで `caseId` と `templateKey` を受け取り、Supabaseから案件情報を取得 → Claude API（claude-sonnet-4-20250514）で書類を生成 → documents テーブルに下書き保存。4種のテンプレート対応：遺産分割協議書・相続人調査報告書・財産目録・送付状 |
