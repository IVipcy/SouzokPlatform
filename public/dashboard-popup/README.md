# ダッシュボード達成ポップアップ画像

このフォルダには以下の3画像を配置してください（必須）:

| ファイル名 | 用途 |
|------------|------|
| `neutral.png` | 真顔（初期フレーム、達成判定中の演出） |
| `celebration.png` | 笑顔・拍手・クラッカー（目標達成時） |
| `angry.png` | 怒り顔（目標未達時） |

## 推奨仕様
- 形式: PNG（背景透過推奨）
- アスペクト比: 縦長（3:4 程度）
- 短辺 600px 以上 / 短辺 1000px 以下推奨
- ファイルサイズ 200KB 以下推奨（圧縮済）

## 配置先
`public/dashboard-popup/neutral.png` のように public 配下に置けば、
Next.js から `/dashboard-popup/neutral.png` でアクセスできます。
